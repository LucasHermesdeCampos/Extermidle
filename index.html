Como Jogar

  • Objetivo: Descubra o personagem do dia em até 6 tentativas.
  • Digite um nome: Use a barra de pesquisa para encontrar personagens.
  • Cores das tags:
    • ■ Verde = Correto
    • ■ Vermelho = Incorreto
    • ■ Cinza = Desconhecido
  • Idade: Setas (↑/↓) indicam se o alvo é mais velho ou mais novo.
  • Atributos: Tags com descrições básicas do personagem como status e vida.
  • Dicas: Ocupações, Associações e Aparições são verificados individualmente.
  • Vidas: Você perde uma vida a cada chute errado.
  • Descrição: Passe o mouse sobre as tags para ver descrições dos atributos.
